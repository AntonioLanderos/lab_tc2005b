¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Diferencias: Java es un lenguaje de programación orientado a objetos que se compila y ejecuta en una máquina virtual (JVM). JavaScript es un lenguaje de scripting interpretado que se ejecuta en navegadores web. Semejanzas: Ambos utilizan una sintaxis similar y son orientados a objetos.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

getDate(), getDay(), getFullYear(), getHours(), getMinutes()

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

push(), pop(), shift(), unshift(), slice()

¿Cómo se declara una variable con alcance local dentro de una función?

Usando la palabra clave let o const dentro de la función.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Puede causar efectos secundarios no deseados, dificultar la depuración y aumentar la posibilidad de colisiones de nombres.
