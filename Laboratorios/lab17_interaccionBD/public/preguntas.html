¿Qué ventajas tiene escribir el código SQL únicamente en la capa del modelo?

Centraliza el acceso a datos, facilita el mantenimiento, mejora la seguridad, permite la reutilización del código y promueve una clara separación de responsabilidades, resultando en un código más limpio y organizado.

¿Qué es SQL injection y cómo se puede prevenir?

es un tipo de ciberataque encubierto en el cual un hacker inserta código propio en un sitio web con el fin de quebrantar las medidas de seguridad y acceder a datos protegidos. restringir los procedimientos de la base de datos, sanear las entradas de la base de datos y aplicar el acceso con menos privilegios.
